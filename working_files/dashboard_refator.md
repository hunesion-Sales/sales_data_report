1. 설정>산업군 관리 메뉴 추가
    - 산업군 등록/수정/삭제 
    - 입력한 데이터의 아래 항목에 따라서 산업군 할당 기능
     - 할당 된 산업군에 따라서  매출, 매입, 매출 이익 데이터를 통한 통계치

2. 설정>데이터 관리 기능 추가
	1) 산업군별 데이터 업로드 기능 - 기존의 제품별, 부문별 엑셀 데이터 일괄 업로드 로직을 적용하여  “산업군별 데이터” 버튼 추가 및 기능 구현 
          2) 수주잔액 데이터 업로드 기능 - 제품별, 부문별, 산업군별 수주 잔액 (현재 월(말일 이전의 경우) or 현재월 +1월 부터 ~ 12월까지 예상되는 매출/매출이익 데이터)데이터 파일 업로그 기능 추가 구현 (별도 데이터 베이스로 관리 )
              - 지난 달의 데이터는 무시 할 것
              - 해당 데이터는 대시보드의 수주잔액 항목 데이터로 사용

3. 제품 마스터 관리 기능 수정
     - 첨부한 파일 제품군으로 제품명 할당
     - 제품군으로 대시보드의 통계치로 사용

4. 설정>목표관리에 “제품별 목표치 설정” 기능 추가
     - 기존 영업부문 목표관리와 동일하게 제품별 목표 설정 기능 추가

5. 대시보드 리팩토링
     - 첨부파일의 설계대로 리팩토링 제공

 1)  대시보드 및 다른 페이지에서도 공통적으로 적용 되어야 하는 기준 (중요함)
   a. 데이터 로딩 기준(반드시 지킬것) : 상단의 매출액 보기, 매출이익 보기 에 따라서 대시보드의 데이터는 매출액, 매출 이익으로 변경
   b. 데이터 업로드 기준 일자 : 설정>데이터관리 에서 데이터 업로드 일자 
    c. 연도, 월별, 분기별, 반기별 선택 기능 - 선택한 기준으로 대시보드의 실적 데이터 집계 기준으로 사용

    2) 실시간 매출 현황 (통계 기준 : 제품별 업로드 데이터)
       - 실적 : 해당 기간 동안의 실적 데이터의 매출 혹은 매출이익 의 합계 데이터 (백만원 단위)
       - 연간 달성율 : 연간 데이터의 매출 혹은 매출 이익 총합 대비 실적 데이터(매출액, 매출이익액)의 달성율 (%)
       - 수주 잔액 : 데이터 관리의 “수주잔액” 업로드 데이터 에서 총 매출 혹은 매출이익 (백만원 단위)
       - 연간 예상 실적 : “실적” 데이터와 “수주잔액” 데이터를 합산한 총 매출 혹은 총 매출이익 (백만원 단위)
       - 연간 달성율(예측) : 연간 총 매출액과 총 매출이익 대비 “연간 예상 실적”의 달성 비율(%)

     4) 월별 실적 및 예측 현황(상단에서 선택한 “매출액” 혹은 “매출이익” 으로  항목 표시 그래프, 데이터도 해당 선택한 데이터 참조) - 매월 매출 혹은 매출 이익을 월별로 표시
        - 누적 실적 선택 시 :  그래프와 데이터는 전월 실적 + 당월 실적+당월 수주 잔액 을 더하여 표시
        - 월별 실적 선택 시 :  그래프와 데이터는 당월 실적+당월 수주 잔액을 더하여 표시
        - 월별 “제품별” 데이터의 2025년(전년도), 2026년(당해년도)의 실적, 수주 잔액을 월별로 표시
        - 달성율은 : 연 목표 대비 해당 월의 실적 혹은 실적+수주잔액 의 달성율 그래프
        - 마우스를 올리면 해당 월의 실적, 수주잔액, 달성율 표시 (백만원, %) 

       5) 주요제품별 실적 순위 현황(상단에서 선택한 “매출액” 혹은 “매출이익” 으로 항목 표시, 그래프, 데이터도 해당 선택한 데이터 참조)
        - 설정>제품 마스터 관리의 제품군을 기준으로 X 축 항목 사용, 2026년 실적이 가장 높은 순서대로 배치
        - 제품군별 2025년 실적, 2026년 실적, 수주잔액을 그래프 와 데이터 표시 
        - 달성율 및 마우스 올릴때 기능은 4) 월별 실적 및 예측 현황 과 동일하게 구현 (공통 모듈로 이용 할것)

      6) 주요 산업군별 누적 실적 순위 현황 (상단에서 선택한 “매출액” 혹은 “매출이익” 으로 항목 표시, 그래프, 데이터도 해당 선택한 데이터 참조)
        - 설정>산업군 관리의 산업군을 기준으로 X 축 항목 사용, 2026년 실적이 가장 높은 순서대로 배치
        - 산업군별 2025년 실적, 2026년 실적, 산업군별 수주잔액을 그래프 와 데이터 표시 
        - 달성율 및 마우스 올릴때 기능은 “4) 월별 실적 및 예측 현황” 과 동일하게 구현 (공통 모듈로 이용 할것)
 
    7) 부문별 매출(혹은 매출 이익) 및 달성률
      * 실적만 보기, 실적+수주 잔액 으로 보기 선택 기능 추가
       - X 축 항목은 기존대로 유지
       - 각 부문별 실적(매출 혹은 매출이익) + 부문별 수주 잔액 데이터를 그래프로 표시 
       - 위 선택 항목에 따라서 달성율 변경하여 표시(2026년 부문별 총 매출액, 총 매출 이익 대비)
